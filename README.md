# Credit Card Fraud Detection Dashboard

An interactive dashboard built with R Shiny for exploratory data analysis and prediction on churn numeric dataset.

## Technologies
- **Language**: R
- **Framework**: Shiny

## Purpose
Perform EDA and predictive modeling on credit card churn datasets with interactive visualization and model training.

## Features
- Exploratory Data Analysis (EDA)
- Churn prediction models
- Data upsampling and undersampling
- Interactive Shiny dashboard
- Real-time model training
- Data visualization

## Usage
Run the R Shiny app to explore the dataset and train predictive models with different sampling techniques.

## Dataset
Works with numeric churn datasets for credit card fraud/churn analysis.
